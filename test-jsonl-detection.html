JSONL Detection Test

이 파일은 JSONL 감지 기능을 테스트하기 위한 샘플 데이터를 제공합니다.

테스트 시나리오:

  1. JSON 모드에서 아래의 JSONL 데이터를 입력
  2. 파싱 오류가 발생하면서 JSONL 감지 경고가 표시되는지 확인
  3. "JSONL로 변경" 버튼을 클릭하여 자동으로 JSONL 모드로 전환되는지 확인

테스트용 JSONL 데이터:

{"name": "홍길동", "age": 30, "city": "서울"}
{"name": "김철수", "age": 25, "city": "부산"}
{"name": "이영희", "age": 28, "city": "대구"}
{"name": "박민수", "age": 32, "city": "인천"}

예상 동작:

  • JSON 모드에서 위 데이터 입력 시 파싱 오류 발생
  • 오류 메시지 하단에 "JSONL 형식이 감지되었습니다" 경고 표시
  • "JSONL로 변경" 버튼 클릭 시 자동으로 JSONL 모드로 전환
  • JSONL 모드에서 정상적으로 4개의 JSON 객체가 트리로 표시
